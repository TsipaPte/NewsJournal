# NewsJournal

News Journal - это сайт с новостями выполненный на языках HTML и CSS. Сайт состоит из 4 главных элементов - шапки, секции "О нас", секции с последними новостями и footer`а сайта. Также у данного сайта есть адаптивная вёрстка. В некоторых местах используются семантические теги.

## Инструкция по запуску

### Инструкция для Windows

1. Скачайте проект
2. Запустите файл `index.html` в браузре, для этого дважды кликнете на файл <br>
  2.1. Файл не открылся в браузере? Нажмите на него `ПКМ` и выберете пункт `Открыть с помощью`, а далее выбераите браузер
  **Обратите внимание, что пункт `Открыть с помощью` может отличаться от вашего языка системы**
